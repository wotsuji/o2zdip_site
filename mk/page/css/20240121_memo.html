よく使うような項目をメモしました。

◆viewport＋medhiaクエリ

・viewportを使うにはheadにmetaタグ追加が必要


<meta name=”viewport” content=”width=device-width,initial-scale=1″>
「width=device-width」表示領域の幅を指定するという意味
「initial-scale=1」初期ズーム倍率（1を指定しておくと初期表示が安定する）

・メディアクエリ

@media screen and (min-width: 420px) {
    /* 420px - */
    .bg-g { /* class指定の場合 */
        background-color:#a1a1a1;
    }
}

・横幅の指定（後の記述で上書きしていく）

min-width: 0px;      # 0-420
min-width: 420px;  # 420-700
min-width: 700px; # 700-900
min-width: 900px; # 900-1200
min-width: 1200px; # 1200-

min-width ○pxから適用させる
max-width 最高○pxまで適用させる

◆position

position: relative; は現在位置を基準に、相対位置を指定する。position: absolute; とセットで各要素に使う。
positionプロパティは「top」「right」「bottom」「left」の4つのプロパティと合わせて要素の配置を指定する。

◆スクロールに要素を追随させる（ヘッダなどで使う）


.some-component{
    position: sticky;
    top: 0;
} 
.main-footer{     
     position: sticky; 
     bottom: 0;
}

◆display

block⇒要素が横までいっぱいに広がり、縦に並んでいく
inline⇒要素が平ぺったく横に並んでいく
inline-block⇒blockとinlineの中間（高さ・上下マージン/パディングを指定できる）
none⇒非表示になる

◆ビューポートの％指定

VW（ビューポートからの横幅パーセント）


width 100VW; /* 親要素の影響を受けない */
width 100%; /* 親要素に影響を受ける */

◆奇数番目 


li:nth-child(3) 3つ目
li:nth-child(2n) 2の倍数
li:nth-child(3n+1) 3の倍数を1つずらして、1,4,7,10

◆ページ内herfでスルスル動くスクロール（下記を加えるだけ）


html {
    scroll-behavior: smooth;
}

◆カスタムプロパティ


:root {  /* 疑似要素：rootはhtmlタグと同位と考えていい */
    --main-color: #4dblec; /* CSS変数を宣言する */
}
a {
    color: var(--main-color); /* var()を使ってCSS変数を使用する */
}

◆トランジション（:hoverと通常時のbackgroundをゆっくり変化させる）


.transition-1s {
    transition: background 1s; # 1秒かけて変化させる
}
.transition-1s { # 全部の変化
    transition: 1s; # 1秒かけて変化させる
}

◆CSSのクラスを付けたりはずしたりできる toggle（JS）


this.classList.toggle('radius');
width 0;
width 100%; /* 親要素に影響を受ける */

◆scale関数で要素を拡大する


/* itemにマウスを重ねた時 */
.item:hover {
    filter: grayscale(0);    /* グレースケール*/
    transform: scale(1.1); /* 大きさの変更 */
    z-index: 3;
    position: relative;
}
