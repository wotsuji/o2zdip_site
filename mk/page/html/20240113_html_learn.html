HTML5を学びなおす

HTML5などで追加されたタグ、最近よく見かけるようになったHTMLタグ
ちょっと見直さないといけないなって思っていたので改めて整理する。
※あくまで個人的に気になった点、おさえておこうかなと思った点を整理しています。

◆labelタグ

入力項目と文字列(ラベル)を関連付ける
<label for="sei"...>姓</label>
<input type="" name="sei"...

◆input type=text に対する「pattern」属性

入力値のパターンマッチ（正規表現）でフォームサブミット時に入力チェック＋エラーメッセージを作成できる。
※HTMLで入力チェックしたとしても、常にサーバーサイドでの入力チェックは必要。
例：
半角英数：^[0-9A-Za-z]+$
郵便番号：¥d{3}-¥d{4}

◆構造タグ

<header><!-- ヘッダ情報 --></header>
<nav><!-- ナビゲーション情報 --></nav>
<main><!-- メインコンテンツ --></main>
<section><!-- セクション(章) --></section>
<aside><!-- 余談/挿話(サイドバー) --></aside>
<footer><!-- フッタ情報 --></footer>
<code><!-- プログラムコードを意味する（固定幅フォントになる） --></code>

◆linkタグに対する「preload」について

ScriptファイルとCSSをブラウザのレンダリングより前に先読みする
<link rel="preload"...
　※reloadを付ける時は「as」を必ず付ける（先読みしたリソースの利用先を指定する）

◆Scriptタグに対する「async / defer」について

　async属性：HTMLパースを待たずにダウンロード完了直後にScriptが実行される
　defer属性：HTMLパースの後にScriptが実行される（タグの記述順に実行される）
　どちらも無い場合は「直列ダウンロード」となる。
　どちらかを付ける場合は「並列ダウンロード」が行われる。
<head>
  <meta charset="utf-8" />
  <title></title>

  <link rel="preload" href="style.css" as="style" />
  <link rel="preload" href="main.js" as="script" />

  <link rel="stylesheet" href="style.css" />
</head>
<body>
  <script src="main.js" defer></script>
</body>
「link rel は preload を付ける」＋「script は defer を付ける」がベストかなと思います。

◆「title属性」

　カーソルを当てると吹き出しを表示する
　※ALT属性は画像の説明（ALTはできるだけ記載する方がSEO的に良い）

◆「 タグ名 data-*=value 」ユーザ拡張属性

　*の部分に好きな名前を指定してユーザ独自の属性を追加することができる
